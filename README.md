## About the Project

This project is a responsive landing page for a restaurant website.
It focuses on clean UI, simple user interactions, and basic frontend logic.

## Features
- Interactive cart
- Reservation form
- Responsive design
- Success message logic

Built with:
- HTML
- CSS
- JavaScript

## Design & Credits

https://www.figma.com/community/file/1568363032267680927/responsive-food-ordering-app
